1. Take a backup of WorkBC_CareerSearchTool database.
2. Download all the files from https://github.com/bcgov/workbc-cst/tree/master/SSOT.
3. Copy all the above files into the following folder path (create the folder structure if not present): C:\src_cst\SSOT
4. Connect to WorkBC_ CareerSearchTool database.
5. Run the SQL scripts in the sequence provided below:
	a. DeleteCSTData.sql
	b. UpdateEducationLevels.sql
	c. UpdateNOCWith2021Data.sql
	d. AddkeysCST.sql
	e. UpdateNOCWithNewIncome.sql
	f. UpdateCommonJobTitlesWithNOC2021.sql
	g. UpdateGeographicalRegions.sql
	h. UpdateOccupationGroups.sql
	i. UpdateNOC2021OccupationGroup.sql
	j. UpdateIndustriesCST.sql
	k. UpdateJobOpening.sql		

6. Share results by running the following queries:
	a. SELECT * FROM [dbo].[NOC]
	b. SELECT * FROM [dbo].[NOCOccupationGroup]
	c. SELECT * FROM [dbo].[NOCOccupationInterest]
	d. SELECT * FROM [dbo].[NOCVideos]
	e. SELECT * FROM [dbo].[CommonJobTitle]
	f. SELECT * FROM [dbo].[EducationLevel]
	g. SELECT * FROM [dbo].[GeographicArea]
	h. SELECT * FROM [dbo].[Industry]
	i. SELECT * FROM [dbo].[JobOpening]
