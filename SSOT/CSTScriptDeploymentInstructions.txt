1. Take a backup of WorkBC_CareerSearchTool database.
2. Download all the files from https://github.com/bcgov/workbc-cst/tree/master/SSOT.
3. Copy all the above files into the following folder path (create the folder structure if not present): C:\src_cst\SSOT
4. Connect to WorkBC_CareerSearchTool database.
5. Run the SQL scripts in the sequence provided below:

	4)UpdateNOCWithNewIncome.sql
	5)UpdateIndustries.sql
	6)UpdateGeographicalArea.sql
	7)UpdateCommonJobTitle.sql
	8)UpdateOccupationGroup.sql
	9)UpdateNOCOccupationGroup.sql
	10)UpdateJobOpening.sql	
	11)UpdateOccupationalInterest.sql
	12)UpdateNOCOccupationalInterest.sql
	13)UpdateNOCVideos.sql
	14)AddFKNOC.sql
